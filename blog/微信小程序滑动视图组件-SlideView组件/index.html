微信小程序滑动视图组件-SlideView组件

2023年7月24日 · 阅读需 5 分钟
冯雪涵
阅己 悦己 越己

在小程序开发中，滑动页面是一种常见且强大的交互方式。本文将介绍小程序中的SlideView组件，该组件是一款专门用于小程序中实现滑动页面效果的高级组件。它基于小程序原生的滑动事件，提供了丰富的配置选项和灵活的扩展能力，而且简单易使用。

我们在开发过程中小程序实现了列表的展示，同时还要实现列表的左滑删除，如下图的效果，这时我们可以使用WeUI组件库里面的Slideview组件来帮助我们快速实现效果。

实现步骤：

  1. 首先确保我们已经在小程序项目中引入了微信官方的WeUI组件库，在我们的app.json文件下指定引用的扩展库
 "useExtendedLib": {
    "weui": true
  },
  1. 然后在需要使用滑动删除页面的json配置文件中添加以下代码：
{
  "navigationBarTitleText": "列表",
    "usingComponents": {
      "mp-slideview": "weui-miniprogram/slideview/slideview"
    }
}
  1. 核心内容

  • 在小程序的wxml文件中使用Slideview组件，用 mp-slideview组件 包裹住所有的列表，这样所有的列表，即每个item都可以实现左滑删除的功能。我们在向左滑动时右侧会出现按钮，同时我们需要对这个按钮buttons进行设置(第二段代码)，比如设置按钮的个数、类型、文字、图标等，具体见第一段代码。

  • mp-slideview组件第一行属性wx:for等是为了遍历出列表，这块没啥特殊意义；

  • 第二行是为了实现列表左滑的效果，show="{{false}}"是为了显示slideview，也就是显示左滑后的右边按钮区域，buttons="{{slideButtons}}"是为了设置左滑显示的按钮组，具体代码是第二段的代码，bindbuttontap="slideButtonTap"是点击buttons按钮触发的事件，这里可以获取索引，比如我们点击的是第几个列表啊，然后进行下一步操作，icon是为了让按钮显示图片，同时需要在buttons里面具体设置src，图片路径，按钮默认是显示文本的，同时我们为删除按钮绑定了点击事件 bindbuttontap="slideButtonTap"，能处理删除的具体操作；

  • 做完以上两步后，我们发现每个列表都可以向左滑动，显示删除按钮了。但是所有的列表可以同时向左滑动，我们平时看到的列表滑动效果是只能一个向左滑动，当第二个滑动的时候，第一个恢复原状。第三行就解决这个问题了，bindshow="onSlideShow"为列表添加按钮显示触发事件，当有列表显示了，我们记录下它当前的索引，然后 show="{{plant.id == slideIndex}}" 来判断当前滑动的索引是否和遍历的索引相等，也就只让当前滑动的那一条列表显示了slideview，让其显示，其他的滑动后slideIndex值变了，上一条滑动的恢复原状，当前的显示。

  //整体框架
  <mp-slideview  wx:for="{{plants}}" wx:for-item="plant" wx:key="{{plant.id}}"  data-index="{{plant .id}}"
  show="{{false}}" buttons="{{slideButtons}}" bindbuttontap="slideButtonTap" icon
  bindshow="onSlideShow" show="{{plant.id == slideIndex}}">
    <!-- 下面的是小程序的内容列表部分，不重要 -->
    <navigator  url="/pages/list_details/list_details">
        <!-- 文字部分 -->
        <view class="content">
            <ul>
            <li>
                <span>列表1</span>
            </li>
            <li>
                <span>列表2</span>
            </li>
            <li>
                <span>列表3</span>
            </li>
            <li>
                <span>列表4:</span>
            </li>
            <li>
                <span>列表5</span>
            </li>
            </ul>
        </view>

        <!-- 图片部分 -->
        <view>
            <image class="item_image" src="http://up.deskcity.org/pic_source/2f/f4/42/2ff442798331f6cc6005098766304e39.jpg"></image>
        </view>
    </navigator>
  </mp-slideview>
    //滑动列表的按钮设置
    data:{
        slideButtons: [{
            src: '/image/delete.png',
            type: 'warn',
    }, ]
    }
  //点击列表删除按钮,点击事件
  slideButtonTap(e) {
    let index = e.currentTarget.dataset.index;
    this.setData({
      cilckIndex: index
    })
  },
  //滑动列表：保证只能同时有一个列表的左滑菜单显示
  onSlideShow(e) {
    let index = e.currentTarget.dataset.index
    this.setData({
      slideIndex: index,
    })
  },

综上，我们现在可以完完整整的使用SlideView组件实现了左滑删除效果
